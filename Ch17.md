# Ch17 输入设备应用编程

## 17.1 输入类设备编程介绍

### 17.1.1 什么是输入设备

鼠标、键盘、触摸屏、遥控器、画图板等。用户通过输入设备与系统交互。

### 17.1.2 input 子系统

Linux 通过 input 子系统统一管理输入设备上报的不同数据类型。

驱动开发人员基于 input 子系统开发输入设备的驱动程序，input 子系统可以屏蔽硬件的差异，向应用层提供一套统一的接口。
