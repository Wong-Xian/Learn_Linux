# 5.2 stat函数 {ignore=true}

stat命令的底层是调用stat()函数
stat()函数是Linux中的系统调用
可通过man 2 stat命令查看

``` c
#include<sys/types.h>
#include<sys/stat.h>
#include<unistd.h>

int stat(const char* path, struct stat* buf);
```
path指定了一个需要查看属性的文件
buf指向了一个记录文件属性的结构体
返回值：成功返回 0；失败返回-1，并设置 error。
